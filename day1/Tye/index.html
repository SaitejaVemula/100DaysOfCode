  • The code which is stored in Git keeps changing as more code is added. Also, many developers can add code in parallel. So Version Control System helps in handling this by maintaining a history of what changes have happened.

Basic Git Commands

git --version

is used to verify if Git is installed

cd repository-name

git init

adds a local Git repository "repository-name"

git add index.html

staging the file "index.html"

git add .

if you want to add all the files to the staging area

git status

used to find out information regarding what files are modified and what files are there in the staging area

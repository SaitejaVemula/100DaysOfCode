Basics of Git

What is Git?

Git is a distributed version-control system for tracking changes in any set of files, originally designed for coordinating work among programmers cooperating on source code during software development.

Basic Git Commands

To Create a new repo

git init

To see the status of the repo

git status

To add a file to the staging area

git add [FILENAME]

To add all files to the staging area at once

git add .

To commit the changes to repo

git commit -m "SHORT DESC ABOUT CHANGES"

To list all branches of a repo

git branch

To Create a new branch

git branch [BRANCHNAME]

To view the logs

git log

To enter/checkout a branch or to a previous log

git checkout [BRANCHNAME / LOGID]

To merge Branch A into Branch B (you need to checkout into Branch B first)

git merge [BRANCH-A]
